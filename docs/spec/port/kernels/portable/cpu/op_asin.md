> Concretely: validates `out` is a floating type (`tensor_is_floating_type` →

> `std::asin((double)val_in)`; otherwise `std::asin((float)val_in)`; cast to
> CTYPE_OUT. IEEE `std::asin` semantics: domain `[-1, 1]`, range `[-pi/2, pi/2]`;
> inputs outside `[-1, 1]` (and NaN) yield NaN. Returns `out`.
